# 🐳 DevOps Docker Practice Repository

This repository contains hands-on practice for **Docker, Custom Images, and Docker Compose**.

It includes multiple applications built using different technologies and containerized using Docker.

---

## 🚀 What This Repo Covers

- 🐳 Writing Dockerfiles for different stacks  
- 📦 Creating custom Docker images  
- 🔗 Multi-container setups using Docker Compose  
- ⚙️ Full-stack containerized applications  

---

## 🛠️ Tech Stack

- Docker 🐳  
- Docker Compose  
- Nginx  
- Node.js  
- Python (Flask)  
- React.js  

---

## 🐳 Custom Images

### 🔹 Nginx App
- Serves static HTML using Nginx  
- Demonstrates basic Dockerfile setup  

### 🔹 Node App
- Simple Node.js server  
- Shows backend containerization  

### 🔹 Python App
- Flask-based backend  
- Demonstrates Python Docker image  

### 🔹 React App
- Frontend React application  
- Built and served via Docker
